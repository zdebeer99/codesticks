List2code Documentation

List2code is a simple text generator that can be used to generate text from a comma delimited or tab delimited table.

Simply copy and paste your data from Excel or note pad into the 'Data' field, create a template and click on generate.

The template will repeat for each line in your table, and replace the place holders {0},{1},... with the value contained in that column position.

Data Syntax

Comma or Tab delimited.

Escaping

To escape a comma or a tab in a column wrap the text with double quotes.

Ex: apple,"keep,in,one,placeholder",column3 result:
{0} = apple
{1} = keep,in,one,placeholder
{2} = column3

Template Syntax

Escaping

Escaping a template variable is done by adding the '@' character in front of column placeholder in the template.

Examples;

Ex 1: "@{0}" result: "{0}"

Ex 2: "Escape this @{0} but not this '{0}'" result: "Escape this {0} but not this 'value'"

to add a @ character in your code, you will need to escape the escape char.

Examples;

Ex 1: "@@" result: "@"

Ex 2: "someone@@mail.com" result: "someone@mail.com"
